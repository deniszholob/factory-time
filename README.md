# Factory Time [Website](https://factorytimegame.com)

A factory automation incremental game at the end of the universe. Start crafting by hand, automate production, and build the Chronoforge to escape.

- [Play Demo on Steam](https://store.steampowered.com/app/4021110/Factory_Time/)
- [Play Prototype on Itch.io](https://deniszholob.itch.io/factory-time)
- [Website](https://factorytimegame.com)
- [Website GH](https://deniszholob.github.io/factory-time/)
- [Discord](https://discord.gg/PkyzXzz)

```
- Build the last factory in the final pocket of usable energy before the heat death of the universe.
- Build production chains, automate everything, and get powerful upgrades.
- Create, share, and play custom scenarios.
```
